# 📝 To-Do List App (v1.0.0)

A simple To-Do List web application built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

- Add new tasks
- Edit tasks
- Delete individual tasks
- Delete all tasks
- Filter tasks:
  - All
  - Completed
  - Not Completed

---

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (DOM Manipulation)

---

## 📂 Project Structure

```
To-Do List App
├─ index.html
├─ styles.css
├─ script.js
```

---

## ⚠️ Known Issues

- ❌ Uses inline event handlers (`onclick`)
- ❌ No data persistence (data lost after refresh)
- ❌ Repeated IDs in DOM (invalid HTML)
- ❌ Performance issues (full re-render)
- ❌ Potential XSS vulnerabilities (`innerHTML` usage)
